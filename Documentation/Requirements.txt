Task:

Design a REST API for creating and retrieving users. A user has:

- a unique username
- a name
- a date of birth

Requirements:

1. When creating a user, it must be verified that the user is at least 16 years old.

2. The user should be persisted (technology optional).

3. After successful creation, an event should be sent via RabbitMQ, informing other services about the new user.

Expected result:

A complete system is not expected. Please provide:

- an architectural overview (briefly sketched or as a diagram)
- domain and service models
- an API design (e.g., endpoints, examples)
- a partial implementation

Programming language and frameworks:

Framework: .NET 10, ASP.NET Core
Programming language: C#
Persistent layer/database: Optional

===========================================================================

Aufgabe:
